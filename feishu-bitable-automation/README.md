# AI Video Factory Bitable Automation

This is a Feishu Bitable automation action. It calls the FastAPI backend in the parent project and creates a video generation task from the triggering Bitable record.

## Configure

Before uploading, replace the placeholder in `block.json`:

```json
{
  "blockTypeID": "blk_xxx"
}
```

Use the `blockTypeID` generated by the Feishu Open Platform capability for Bitable automation actions, not the cloud document widget block ID.

## Install

```bash
npm install
```

## Build

```bash
npm run build
```

## Upload

```bash
npm run upload -- -v 1.0.0 -d initial
```

After uploading, go to Feishu Open Platform, create a Bitable automation action version, publish the app version, then return to Bitable automation and select this action version.

## Automation Usage

Add this action after a record trigger. Configure:

```text
后端服务地址: https://your-api.example.com
记录 ID: leave empty when using a record trigger, or quote a previous step record ID
```

The backend must be public HTTPS in production. Feishu automation cannot call `localhost`.
